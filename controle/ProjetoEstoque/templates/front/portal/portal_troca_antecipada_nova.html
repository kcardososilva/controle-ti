Como funciona
Por que adiantar o substituto

Sem a troca antecipada, o equipamento com defeito fica parado na fazenda até o reparo ser combinado. Adiantando o substituto, a operação não para — o defeituoso só sai depois que o novo já chegou.

O substituto entra no sistema como Pausado assim que a troca é aberta. Ele passa a valer como equipamento em uso quando o TI confirma o recebimento na fazenda — e é só a partir daí que o defeituoso é enviado para vocês, com prioridade.

O modelo informado aqui é o que identifica o equipamento no estoque; não é preciso preencher um nome separado.

Cada equipamento aceita uma troca em aberto por vez — se já houver uma ordem pendente para ele, ele não aparece na lista de seleção.

Para acompanhar o andamento de uma troca já aberta, use a lista de Troca Antecipada.
